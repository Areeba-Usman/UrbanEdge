# UrbanEdge
A clean, responsive e-commerce frontend built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just smooth shopping experience.

✨ Features

🏠 Multi-page SPA — Home, Shop, and Contact pages without reload
🛒 Fully Functional Cart — Add, remove, update quantity, and clear cart
💾 Persistent Cart — Cart data saved in localStorage across sessions
🔍 Live Search — Instant product search as you type
🏷️ Category Filter — Filter products by Watches, Audio, Shoes, Bags
🌙 Dark / Light Theme Toggle — Eye-friendly theme switch
📱 Responsive Design — Works on mobile, tablet, and desktop
📬 Contact Page — Simple contact form with business info
